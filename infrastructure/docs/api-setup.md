# API setup

APIs are implemented by workspaces prefixed by `api-`:

- [api-admin](../../api-admin/)
- [api-auth](../../api-auth/)
- [api-public](../../api-public/)
- [api-stripe](../../api-stripe/)
- [api-user](../../api-user/)

To deploy an API, for instance `api-user` launch script

```sh
npm run deploy:api:user
```
